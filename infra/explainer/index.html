What is Worldpool?

Worldpool is an open-source threat intelligence project that maintains a continuously updated list of open proxies on the internet. It helps security researchers, network operators, and developers identify misconfigured or malicious proxy servers that could be used to relay traffic or conduct attacks.

The full source code is publicly available on GitHub → CelestialBrain/worldpool

Why was my IP scanned?

Your IP address falls within a range commonly associated with datacenter infrastructure, cloud providers, or ASNs that historically host open proxy servers. The scanner probed common proxy ports (1080, 3128, 8080) to check whether a publicly accessible proxy is running.

What data was collected?

  • IP address and port number
  • Whether the port was open (TCP connect only)
  • Proxy protocol type (SOCKS4, SOCKS5, HTTP) if confirmed
  • Country and ASN (via offline GeoIP database)

We collect metadata only. We never authenticate, never send real traffic through your host, never exploit any vulnerability, and never store personal data.
